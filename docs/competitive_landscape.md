## The honest novelty claim

Sentinel's differentiation is the **rigor of integration**, not invention. Public
repositories already combine XGBoost + SHAP + fairness tooling + Docker + FastAPI.
The genuine differentiators are:

1. Calibration (reliability diagrams, decision-curve analysis) as a first-class output
2. Conformal / uncertainty quantification on individual predictions
3. Continuous fairness **and** drift monitoring shipped as live features, not one-off plots
4. Clinician validation of reason codes and worklist utility
5. Honest evaluation that refuses inflated, leakage-driven metrics

Canonical pitch: *a crowded problem dominated by mediocre, often-biased models;
Sentinel's value is the rigorously calibrated, fairness-audited, continuously-monitored
decision-support product the literature calls for but rarely ships.*
