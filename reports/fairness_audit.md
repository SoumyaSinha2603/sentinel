# Fairness audit — Phase 3 (subgroup metrics, equalized-odds, calibration)

> Read-only audit of the calibrated production path (@phase1 booster + committed isotonic manifest) across the pre-registered sensitive subgroups on the FROZEN `S_eval` surface. Method and every threshold are fixed in `reports/fairness_prereg.md` (the source of truth); this report presents results. No model trained/registered; the Phase-1 holdout was never loaded. Generated by `sentinel.fairness.audit`.

Versions: sklearn `1.7.2`, lightgbm `4.6.0`. seed=42, bootstrap B=2000 (patient-grouped). git `3ad7e37c4483`. S_eval patient sha256 `71a7a8b69e3b` (matches manifest).

S_eval: N=15,734, positives=1,802, prevalence=0.1145. Primary operating point = 10% capacity (implied threshold 0.1875); 5%/20% reported for sensitivity.

> Sensitive attributes (race/gender/age/payer_code) were used ONLY as slicing labels for subgroup metrics, read from the raw cohort columns. They are never added to any feature frame and the deployed model does not consume them as decision inputs.

> **In-sample optimism — read absolute levels with care. `@phase1` was refit on the full 80% train, which CONTAINS `S_eval` (S_eval is a 20% patient-grouped slice carved from inside that 80% for Phase 2). Scoring S_eval with @phase1 is therefore IN-SAMPLE, so the absolute subgroup AUROCs here (~0.70-0.84) are optimistic versus the honest locked holdout AUROC 0.677 and the ~0.62-0.68 ceiling — they are NOT a discrimination win. This audit's deliverable is the RELATIVE parity between subgroups, where the optimism is largely common-mode; absolute discrimination is not comparable to the holdout figure. This is a known, documented property of the frozen S_eval surface (Phase-2 calibration used out-of-fold S_fit scores precisely to avoid this optimism), not audit leakage.**

**`race`**: no material disparity among powered groups (verdict FLAG (monitor)). Any sub-threshold-but-significant gaps are flagged for monitoring, not called failures.

**`gender`**: no material disparity among powered groups (verdict PASS). Any sub-threshold-but-significant gaps are flagged for monitoring, not called failures.

**`age`**: material disparity found (80+ vs 40-60 on auroc; 80+ vs 60-80 on auroc; 80+ vs <40 on auroc; 80+ vs <40 on tpr; 40-60 vs <40 on tpr; 60-80 vs <40 on auroc; 60-80 vs <40 on tpr). Plausible mechanisms to weigh before any mitigation: differential missingness, subgroup base-rate differences, and feature availability — not evidence of an intentionally biased input, since sensitive attributes are not model inputs.

**`payer_code`**: material disparity found (Missing vs Self-pay on auroc; Medicare vs Self-pay on auroc). Plausible mechanisms to weigh before any mitigation: differential missingness, subgroup base-rate differences, and feature availability — not evidence of an intentionally biased input, since sensitive attributes are not model inputs.

Cross-subgroup AUPRC differences are interpreted as base-rate artifacts (AUPRC scales with prevalence), not bias. Small-subgroup AUROC is high-variance; a suspicious gap is more likely a power/variance artifact than a real effect — read every point estimate with its CI.

## Age disparity — OOF robustness

> Prereg §9 robustness follow-through for the age headline. The S_eval verdicts above are unchanged; this section recomputes age discrimination **out-of-fold** on `S_fit` (each row scored by a model NOT trained on it; ~64k rows, all bands well-powered; disjoint from S_eval and the spent holdout) to separate a real gap from in-sample optimism. Generated by `sentinel.fairness.age_oof_robustness`.

Provenance: S_fit patient sha256 `e055e3b1bf67`, N=63,807, positives=7,296. Reproduced pooled OOF AUROC **0.6713** (= W5 0.6713; per-fold 0.656, 0.689, 0.677, 0.664, 0.670). All four bands powered on S_fit: True. seed=42, bootstrap B=2000 (patient-grouped).

> The committed isotonic map was FIT on these S_fit OOF scores in W5, so per-band ECE here is mildly optimistic FOR THE CALIBRATOR (a low-capacity monotone map — small effect). AUROC/AUPRC are calibration-invariant (isotonic is monotone), so they are unaffected. Discrimination is the primary deliverable of this check; ECE is secondary.

### Verdict: **GAP SURVIVES (real)**

Out-of-fold, AUROC declines monotonically across the age gradient (<40 0.751 → 40-60 0.698 → 60-80 0.659 → 80+ 0.623). The material signal is this CUMULATIVE age gradient, not an 80+ cliff: the adjacent 80+ vs 60-80 step is sub-threshold (Δ=-0.036 CI [-0.055, -0.018] — statistically significant but |Δ| ≤ 0.05), while the cumulative multi-band gaps clear the bar (80+ vs 40-60 Δ=-0.075 CI [-0.097, -0.052], material; 80+ vs <40 Δ=-0.128 CI [-0.159, -0.096], material). The age finding is REAL and best described as a gradient in rank-separability with age — not a discontinuity at 80+; the plausible mechanism is case mix (elderly risk distributions are compressed, so ranking is intrinsically harder). On calibration, 80+ OOF ECE is 0.006 — far below the 0.05 per-subgroup bar, so 80+ is merely less rank-separable, still well-calibrated. Caveat: this ECE is measured on calibrated OOF scores the committed isotonic map was itself fit on (so it is mildly optimistic FOR THE CALIBRATOR), but that small monotone-map optimism cannot lift 0.006 across the 0.05 bar — the 'well-calibrated, not miscalibrated' conclusion is robust to the caveat.
